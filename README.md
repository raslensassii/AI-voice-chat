# AI Voice Chat

Browser voice chat with AI powered by Web Speech API & OpenRouter.ai.

---

## Features

- Voice recognition & text-to-speech.
- OpenRouter.ai API (OpenAI compatible).
- Simple UI.

---

## Getting Started

### Prerequisites

- Node.js
- OpenRouter.ai API key ([https://openrouter.ai](https://openrouter.ai))
